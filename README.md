# Lab 12 – Searching and Sorting

## Purpose
This lab focuses on:
- searching for values in a list
- sorting data
- understanding how different approaches behave

Detailed instructions are provided in Canvas.

---

## Requirements

Your program should include:
- a linear search function
- a bubble sort function
- test lists (sorted, reversed, random)

---

## Output

- printed results showing search and sorted lists
- answers to reflection questions

Commit your code and submit your repo link on Canvas.
